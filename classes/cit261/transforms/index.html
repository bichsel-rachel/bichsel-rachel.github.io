Transforms

  • The transform property allows a 2D or 3D change to an element
  • In 2D you can rotate, translate, skew, and scale elements.

Rotate

  • Rotates clockwise and counterclockwise in degrees
  • Positive goes clockwise
  • Negative goes counterclosewise
  • Syntax: transform: rotate(#deg);
rotate(90deg);
rotate(-150deg);

Translate

  • Moves element up/down or right/left
  • Positive X goes right, negative X goes left
  • Positive Y goes down, negative X goes up

transform: translate(30px, 20px);

Skew

  • Tilts the X and Y axes of element
  • Positive X tilts left, negative X tilts right
  • Positive Y tilts down, negative Y tilts up
skewX(30deg);
skewY(15deg);
skew(25deg, -10deg);

Scale

  • This allows to increase or decrease size of element
  • Can scale both y and x-axis by using scale()
  • scaleY() is for just scaling the y-axis
  • scaleX() is for just scaling the x-axis
scale(1.5)
scale(1.5, .5)
scaleX(1.5)
scaleY(1.5)

Multiple Transforms

  • You can do multiple transforms in one line. Just add them without commas

transform: skewX(25deg) translate(100px, 75px) scale(.7);
