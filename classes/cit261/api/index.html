Local Storage API

Access Local Storage by using localStorage.

returns a storage object. Storage has one proprty, length.

Methods

setItem(key, value)

adds item to storage. Uses strings for key and value. If string is not passed it will convert to string.

Example: localStorage.setItem('name', 'Bob');

First Name:

  • setItem(key, value)
  • getItem(key)
  • removeItem(key)
  • key(n)
  • clear()

Storage Limits

  • Chrome, IE, Firefox: 10MB
  • Safari: 5MB for local storage

Session Storage

function colors(){ var colors = ['red','orange','yellow','green','blue','purple']; sessionStorage.setItem("colors", JSON.stringify(colors)); document.getElementById('colorsResults').innerHTML = sessionStorage.getItem('colors'); }


Local Storage

function hogwarts(){ var houses = ['Gryffindor','Slytherin','Ravenclaw', 'Hufflepuff']; localStorage.setItem("houses", JSON.stringify(houses)); var retrievedData = localStorage.getItem("houses"); var house = JSON.parse(retrievedData); document.getElementById('hogResults').innerHTML = house; }
