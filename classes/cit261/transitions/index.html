Transitions

Transitions go hand-in-hand with transforms.

Why we need Transitions

.trans:hover { transform: rotate(45deg);
transition: all 1s; }

No Transition
Transition

Two things required for transitions:

  • transition-property
  • transition-duration

transition shorthand:

transition: [property][duration][timing-function][delay];

transition-property

transition-property: property;

  • Specifies what css property the transition will be applied to.
  • Can do one property or do all properties in rule-set
  • Default is all

Example

.snake {
width: 50px;
height: 50px;
border-radius: 25%;
background-color: orangered;
transition-property: width;
transition-duration: 2s;}

.snake:hover{
width: 200px;
background-color: darkgreen; }

transition-duration

transition-duration: time;

  • Specifies time span of transition
  • Can be in seconds or milliseconds

Example

.square1:hover { transform: scale(1.5);
transition: all 300ms; }

300 ms
5 seconds
1 second

Optional Properties

transition-timing-function

Define the speed over the duration of the transition.

  • Default timing is ease
  • Other options are:
    • linear
    • ease
    • ease-in
    • ease-out
    • ease-in-out

Example

.par1:hover {
transition: all 2s linear;
background-color: red;
transform: translateX(200px) skew(20deg); }

delay

Specifies when transition will start.

Example

.circle2:hover {
transition: all 2s 1s;
transform: scaleY(2); }
