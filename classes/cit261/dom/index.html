DOM Manipulation

  • DOM stands for Document Object Model. It is a programming interface.
  • It is created when a web page is loaded.
  • JavaScript the can use the object model to change, add, get, or delete things in HTML to make it more dynamic.
  • HTML elements are objects in DOM. It uses properties (value you get) and methods(action you do).
  • Most common method to use is getElementById
  • An example of a DOM property is innerHTML.

createElement

  • Using getElementById helps control where element and text is inserted.
  • If you use document.body.appendChild("newPara), it will put the new paragraph at the very bottom of the body element.
  • You can also insert text within the newly created paragraph element. This is discussed in next section.

insertBefore

  • appendChild always does it at the end. Insert before lets you insert before other children.
function beforeName() {
var newName = document.createElement("li");
var textnode = document.createTextNode("Hilary");
newName.appendChild(textnode); var list = document.getElementById("partyList");
list.insertBefore(newName, list.childNodes[0]); }

removeChild

  • You can remove any child by specifying childNodes[#].
function removeName() {
var list = document.getElementById("partyList");
list.removeChild(list.childNodes[0]); }



function removeThirdName() {
var list = document.getElementById("partyList");
list.removeChild(list.childNodes[2]); }

replaceChild

  • As with removeChild, you can specify what child to remove with replaceChild. In this cae you use this syntax: node.replaceChild(newnode, oldnode).
function replacePartyList() {
var textnode = document.createTextNode("Ricky");
var item = document.getElementById("partyList").childNodes[0];
item.replaceChild(textnode, item.childNodes[0]); }
