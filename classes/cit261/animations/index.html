CSS Animations

  • Some older browsers need specific prefixes (-webkit-) to understand the animation.

@keyframes

  • You can change from current stye to a new style at certain times
  • You first need to bind animation to element.
  • You do this by giving the aniation a name and then referencing that name in the @keyframes
  • Specify how long the animation will occur with animation-duration.
  • Animation will go back to the original style once done.

#circleEx {
font-size: 50px;
background-color: turquoise;
width: 200px;
height: 200px;
border-radius: 100px;
animation-name: example1;
animation-duration: 7s;
}
@keyframes example1 {
from {background-color: turquoise;}
to {background-color: purple;} }

  • You can use percentages to specify when to transition to next style. Ex: first example, from was 0: and to was 100%
  • You can also change multiple attributes of an element in one keyframe

#squareEx {
background-color: red;
width: 150px; height: 150px;
position: relative;
animation-name: rainbow;
animation-duration: 10s; }

@keyframes rainbow {
0% {background-color:red; left: 0px; top: 0px; width: 150px;}
10% {background-color:orangered; left: 200px; top: 0px; width: 100px;}
25% {background-color:yellow; left: 200px; top: 150px; width: 80px;}
50% {background-color:green; left: 150px; top: 150px; width: 100px;}
75% {background-color:blue; left: 60px; top: 60px; width: 125px;}
100% {background-color:purple; left: 0px; top: 0px; width: 150px;}
}

Other Animation Properties

animation-delay

  • Delays the start of animation

animation-iteration-count

  • Specifies number of times animation should run
  • You can also specify "infinite" to make the animation constantly run

animation-direction

  • Specifies whether animation should run forwards, backwards or alternate cycles
  • alternate - is played forwards and then backwards
  • alternate-reverse - plays backwards then forwards

animation-timing-function

  • Specifies speed curve of animation.
  • Exactly the same as the transitions
  • ease, linear, ease-in, ease-out, ease-in-out, cubic-bezier(n, n, n, n)

Example of the Above Properties

#propertiesEx {
width: 100px;
height: 100px;
background-color: green;
border-radius: 50px;
position: relative;
bottom: 150px;
right: 150px;
animation-name: example2;
animation-duration: 1s;
animation-delay: 2s;
animation-iteration-count: 2;
animation-direction: forwards;
animation-timing-function: linear;
}

@keyframes example2 {
0% {bottom: 150px; right: 150px;}
45% {bottom: -75px; right: 0px; width: 100px; height: 100px; border-radius: 50px;}
55% {bottom: -75px; right: 0px; height: 50px; width: 120px; border-radius: 25px;}
100% {bottom: 150px; right: -150px; height: 100px; width: 100px; border-radius: 50px;}
}

animation-fill-mode

  • CSS animations don't effect element before first keyframe or after last keyframe. You can override it.
  • Specifies a style for a target element when element is not playing.
  • none - default
  • forwards - element retains style values that is set by last keyframe (depends on animation-direction and animation-iteration-count
  • backwards - element has style values set by the first keyframe (depends on animation-direction)
  • both - extends animation properties in both directions

Shorthand

  • takes on the first six animation properties. Doesn't include animation-fill.

animation: example 10s linear 2s infinite alternate

Example of Animation-fill and Shorthand

This is pretty much the same animation as the first with a few alterations. Take note that instead of going back to the beginning position, it takes on the styles of the first keyframe.

#animationFill {
width: 100px;
height: 100px;
background-color: blue;
border-radius: 50px;
position: relative;
bottom: 150px;
right: 150px;
animation: example3 1s 2s 3 backwards ease-out;
animation-fill-mode: forwards;

} @keyframes example3 {
0% {bottom: 150px; right: 150px; background-color: green;}
45% {bottom: -75px; right: 0px; width: 100px; height: 100px; border-radius: 50px;}
55% {bottom: -75px; right: 0px; height: 50px; width: 120px; border-radius: 25px;}
100% {bottom: 150px; right: -150px; height: 100px; width: 100px; border-radius: 50px; background-color: red;}
}
