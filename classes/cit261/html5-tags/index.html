Video Tag

Syntax

<video src="big_buck_bunny.mp4" controls></video>

Must have controls attribute or video is not playable.

Text between video tags will only play if video formats don't work in browser.

Better Format

<video controls>
<source src="big_buck_bunny.mp4" type="vidwo/mp4">
<source src="big_buck_bunny.ogg" type="video/ogg">
</video>

Three supported video formats for video tag: MP4, WebM, and Ogg. Most browsers support MP4.

Below, the video is set to autoplay, loop, and muted.

Other Optional Attributes

  • autoplay
  • loop
  • muted
  • preload
  • height

Audio Tag

The audio tag is very similar to the video tag.

Syntax:

<audio controls>
<source src="wanted_dead_or_alive.m4a" type="audio/mp4">
<source src="wanted_dead_or_alive.m4a" type="audio/ogg">
</audio>

Three supported video formats for video tag: MP3, WAV, and OGG. Most browsers support MP4.

Optional Attributes

  • autoplay
  • loop
  • muted
  • preload

Canvas

The canvas element is used as a blank canvas that can then be drawn on using JavaScript.

It is useful because you can draw things on the web page in real time like graphs, graphics, games, and art.

Always include a id attribute so you can link the canvas to the Javascript. You can use the width and height attributes to set canvas size. If not, it will use default size.

Default size is 300 pixels x 150 pixels

Adding a border can make it easer to see.

Syntax:

<canvas width="" height=""> </canvas>

Sadly, your browser doesn't support HTML5 canvas tag.
